# Reading Workflow

## Goal

Create a learning artifact that helps a reader understand the paper, not merely summarize it. Preserve the paper's logic while adding enough explanation for a motivated beginner.

## Intake

Check:

- Source file path and output folder.
- Desired output language.
- Whether paragraph-by-paragraph translation is required.
- Whether figures, supplements, question logs, and PDF export are required.
- Whether the final artifact may contain copyrighted text. If the output will be public, use summaries and short excerpts instead of full translation.

## PDF Inspection

Use rendered pages to identify:

- Title, authors, venue, year, DOI, and abstract.
- Section order and paragraph boundaries.
- Main figures, tables, captions, and where each is discussed.
- Supplementary materials links or filenames.
- Article layout issues such as two columns, sidebars, or split captions.

## Note Construction

Use this default sequence:

1. Metadata and source links.
2. One-sentence conclusion.
3. Beginner primer for the field.
4. Reading map.
5. Abstract or editor summary translation.
6. Paragraph-by-paragraph translation and explanation.
7. Figures placed in context with figure notes.
8. Glossary.
9. Mechanism or argument chain.
10. Open questions and discussion prompts.
11. Personal reading summary.

## Translation Rules

- Translate natural paragraphs, not arbitrary PDF line breaks.
- Number paragraphs as `P01`, `P02`, etc.
- Preserve technical terms in bilingual form on first use when useful.
- If extraction order is suspicious, compare with page images.
- Separate translation from interpretation.

## Quality Gates

Before finishing:

- Every main section of the paper is represented.
- Every main figure is inserted or intentionally omitted with a reason.
- Every inserted image path exists.
- Figure notes explain the figure's role in the argument.
- Beginner explanations appear at concept bottlenecks.
- The question log does not duplicate earlier explanations.
- If exported to PDF, representative pages have been visually checked.
