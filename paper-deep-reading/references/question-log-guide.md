# Question Log Guide

## Purpose

The question log captures what the learner actually asked while reading. It becomes a review guide and prevents repeated explanations from spreading across the note.

## What To Record

Record knowledge questions:

- Concept explanations.
- Figure-reading questions.
- Method comparisons.
- Mechanism reasoning.
- Project discussion questions.

Do not record purely operational requests unless they reveal reusable learning content:

- "Export this to PDF."
- "Change the formatting."
- "Upload this to GitHub."
- "Move this image."

## Deduplication Rule

If a new question repeats an existing concept:

1. Merge the new angle into the closest existing question.
2. Add a short "new clarification" subsection if needed.
3. Keep the index stable unless renumbering is explicitly requested.
4. Link to the existing explanation instead of writing the same long answer again.

## Suggested Entry

```markdown
## Q03: How should Figure 1A be read?

**Short answer:** ...

**Logic chain:** ...

**Common confusion:** ...

**Related figures:** ![Figure 1A](assets/figures/fig-1a.png)

**Review prompt:** ...
```
