# Examples

These examples are synthetic and safe to publish. They show the shape of the expected output without reproducing copyrighted journal text.

## Inputs

- `example-inputs/deep-reading-request.md`: full paper deep-reading request.
- `example-inputs/figure-focused-request.md`: figure interpretation request.
- `example-inputs/group-discussion-request.md`: R&D group discussion request.

## Outputs

- `example-output/mini-materials-paper-deep-reading.md`: compact Obsidian-style reading note.
- `example-output/mini-materials-question-log.md`: learning question log.
- `example-output/example-paper-deep-reading.md`: minimal structure example.
