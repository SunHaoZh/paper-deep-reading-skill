# Example Request: Full Deep Reading

Use `$paper-deep-reading` to study this journal paper PDF.

Requirements:

- Build one Obsidian-compatible Markdown note.
- Explain the paper for a beginner.
- Translate or explain each natural paragraph.
- Extract and place each main figure near the paragraph that discusses it.
- For every figure, explain what type of figure it is, what each panel does, how to read it, what claim it supports, and why the authors included it.
- Maintain a separate question log.
- Add glossary, mechanism chain, and group discussion questions.
- Export a PDF if the local environment supports it.
