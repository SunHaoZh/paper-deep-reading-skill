# Example Request: Group Discussion

Use `$paper-deep-reading` to prepare a research-group discussion note for this paper.

Do not only ask questions that can be answered by a one-line AI response. Focus on questions that require judgment, experiment design, mechanism comparison, or project strategy.

Output:

- Five candidate discussion questions.
- One strongest question for group discussion.
- Why that question matters.
- What evidence in the paper supports each side of the discussion.
- What follow-up experiment or calculation would clarify the issue.
