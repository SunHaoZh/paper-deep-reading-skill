# Workflow

1. Inspect the PDF with rendered pages.
2. Build the Obsidian folder structure.
3. Extract text and correct paragraph order.
4. Translate and explain paragraph by paragraph.
5. Crop or extract figures.
6. Place figures in context.
7. Write detailed figure notes.
8. Add beginner callouts and glossary entries.
9. Maintain a deduplicated question log.
10. Export to PDF when requested.
11. Validate links and inspect representative pages.

The core principle: the note should help a reader reconstruct the paper's argument from text, figures, and their own questions.
