# Usage

Example request:

```text
Use $paper-deep-reading to study this PDF. Build an Obsidian note, translate it paragraph by paragraph, place every main figure near the explanation, add beginner explanations, maintain a question log, and export a PDF.
```

For a project group:

```text
Use $paper-deep-reading to generate a group-reading note. Add discussion questions that cannot be answered by a one-line AI query.
```

For figure-focused reading:

```text
Use $paper-deep-reading to explain Figure 1 in detail: what each panel is, how to read it, what evidence it provides, and what common misunderstandings a beginner might have.
```

See `examples/example-inputs/` for more reusable request examples.
