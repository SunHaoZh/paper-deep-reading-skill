# Paper Deep Reading

Use this as a fallback Claude Code slash-command prompt when the native skill folder is not available.

Read the canonical skill instructions from `paper-deep-reading/SKILL.md`, then follow the relevant reference files under `paper-deep-reading/references/`.

Task:

Create a beginner-friendly Obsidian deep-reading note for the supplied journal paper PDF or paper text. Include paragraph-level translation or explanation when requested, figures placed in context, detailed figure notes, beginner concept explanations, a deduplicated question log, glossary, mechanism chain, and optional PDF export.

Quality requirements:

- Preserve the paper's logical order.
- Verify figure placement against rendered pages when working from PDF.
- Explain each figure as evidence, not decoration.
- Distinguish measured, calculated, inferred, and proposed claims.
- Avoid publicly redistributing copyrighted full-text translation unless the source license allows it.
